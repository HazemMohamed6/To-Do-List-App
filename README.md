# 📝 Advanced To-Do List App (Java)

A console-based To-Do List application built using Java, focusing on core backend concepts and clean code practices.

---

## 🚀 Features

- Add new tasks  
- Delete tasks by ID  
- Mark tasks as completed  
- Display all tasks  
- Show pending tasks only  
- Prevent duplicate task IDs  
- Input validation and error handling  
- Persistent data using file storage  

---

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)  
- Encapsulation & Abstraction  
- Java Collections Framework (ArrayList)  
- Stream API & Lambda Expressions  
- Exception Handling (Custom & Built-in)  
- File Handling (Read & Write)  

---

## ⚙️ How It Works

- The application runs in the console  
- Users interact through a menu-driven interface  
- Tasks are stored in memory and saved to a file  
- Data is loaded automatically when the app starts  

---

## ▶️ How to Run

1. Clone the repository  
2. Open the project in your IDE  
3. Run the main class  
4. Use the menu to manage tasks  

---
